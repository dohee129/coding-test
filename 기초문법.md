# 자료형

- 정수형(양의 정수, 음의 정수, 0)

```
a=1000 #양의 정수
print(a)

a=-7 #음의 정수
print(a)

a=0 #0
print(a)
```  

- 실수형
```
a=157.93 #양의 실수
print(a) #157.93

a=-1837.2 #음의 실수
print(a) #1837.2

a=5. #소수부가 0일때 0을 생략
print(a) #5.0

a=-7. #정수부가 0일때 0을 생략
print(a) #-0.7
```  
실수형 데이터를 표현하는 방식으로 파이썬에서는 e나 E를 이용한 지수 표현 방식을 이용 (e 다음에 오는 수는 10의 지수부)
ex) 1e9=10의 9제곱
> 지수 표현 방식은 최단 경로 문제에서 도달할 수 없는 노드에 대하여 최단 거리를 '무한(INF)'로 설정
